# 📚 IMS – Timetable & Attendance Management System

A modern **Institute Management System (IMS)** Android prototype focused on **Timetable Scheduling** and **Attendance Management**, built using **Jetpack Compose**.
This project was developed as part of an academic software design and implementation assignment.

---

# ✨ Features

## 🏠 Dashboard

* Centralized navigation hub
* Quick access to modules
* Search-based navigation
* Clean and intuitive UI

## 📅 Time Table Management

* Create and manage class schedules
* Faculty workload handling
* Subject-wise scheduling
* Edit/Delete timetable entries
* Conflict-aware scheduling flow

## ✅ Attendance Management

* Mark daily attendance
* Add remarks/notes
* Subject-wise attendance tracking
* Attendance history and reports
* Filter and analyze attendance records
